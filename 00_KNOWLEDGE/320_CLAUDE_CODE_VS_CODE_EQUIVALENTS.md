*   **Cline (formerly Claude Dev)**: The primary VS Code equivalent. It provides a full agentic sidebar that can read/write files, execute terminal commands, and perform "Sovereign Web Siphoning" (using browser automation).

*   **Claw Code**: An open-source, terminal-based clone of the Claude Code architecture. It can be run directly within a VS Code terminal to provide a "Claude Code CLI" experience while maintaining local file-system sovereignty.
*   **Continue**: A more streamlined sidebar assistant that excels at "Chat-with-Code" and "Tab-to-Complete" workflows.
