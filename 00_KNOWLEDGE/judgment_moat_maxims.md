# The Judgment Moat: Guiding Maxims for the Zero-Click AI Era
*A practitioner's reference for daily strategic decision-making.*

---

## 1. Daily Decision-Making Grid
Before committing resources to any new content, campaign, or technical workflow, run it through this assessment:

```
                  ┌──────────────────────────────┐
                  │   Is the output of this task  │
                  │   summarizable by an LLM?    │
                  └──────────────┬───────────────┘
                                 │
                  ┌──────────────┴──────────────┐
                  ▼                             ▼
               [ YES ]                       [ NO ]
         (Low-Moat Commodity)         (High-Moat Judgment)
                  │                             │
    ┌─────────────┴─────────────┐ ┌─────────────┴─────────────┐
    │  ACTION: Automate / AI    │ │  ACTION: Double Down /    │
    │  Do not compete on scale. │ │  Monetize via Influence.  │
    └───────────────────────────┘ └───────────────────────────┘
```

## 2. The Four Daily Maxims

### Maxim I: Do Not Compete with the Enclosure
> *“If it can be summarized on the search results page, it will be. Do not pay to build Google's answers.”*
* **The Rule:** If a user can get the complete answer from a 3-sentence LLM summary, do not write a 1,500-word article on it. 
* **The Action:** Stop targeting informational queries that have simple factual answers. Target experiential, analytical, and highly contextual queries instead.

### Maxim II: Build the Inimitable Moat First
> *“If your core asset can be scraped, your business model is a lease scheduled for termination.”*
* **The Rule:** Focus on the remaining 35% of human-exposed labor: original primary datasets, direct human relationships, custom tooling, and bespoke craft.
* **The Action:** Allocate 50% of the content/marketing budget to original data generation, interviews, field tests, or proprietary software utilities.

### Maxim III: Cultivate Portable Attention
> *“Traffic is rented; influence is owned.”*
* **The Rule:** Do not optimize for search algorithms that intend to trap users. Build direct conduits to your audience.
* **The Action:** Treat email list size, private community activity (Discord/Slack/Matrix), and direct brand query volume as primary KPIs. Treat referral search traffic as a secondary, volatile bonus.

### Maxim IV: Separate the Workflow from the Wisdom
> *“You are not your content calendar. You are the judgment that directs it.”*
* **The Rule:** Welcome the automation of the 65% of tasks (competitor tracking, keyword research, initial drafting) to free up bandwidth for the 35% (strategic synthesis, human validation, ethical framing).
* **The Action:** Automate ruthlessly using AI agents for execution, but never delegate the final editor/approver role.

## 3. The One-Sentence North Star
> **"You cannot out-produce AI. You can only out-judge it—and that judgment, when made visible and inimitable, is the only thing humans will follow."**
