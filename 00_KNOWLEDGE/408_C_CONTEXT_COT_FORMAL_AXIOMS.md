### 1. Axioms of Parametric Decoupling

Let:
*   $\mathcal{X}$ be the input space, containing a novel context rule set $R_{\text{novel}}$ and a query $Q$.
*   $\mathcal{A}_{\text{param}}$ be the attention allocation directed toward pre-trained parametric patterns in the model $\mathcal{M}_\theta$.
*   $\mathcal{A}_{\text{context}}$ be the attention allocation directed toward the novel tokens in the active context window.
*   $\mathcal{H}(\mathcal{M})$ be the Shannon entropy of the output sequence distribution.

#### Axiom I: Parametric Dominance (The Gravity Well)
For any standard generation model $\mathcal{M}_\theta$ initialized with a massive pre-training corpus $\mathcal{D}_{\text{pre}}$, if a task query $Q$ contains terms that correlate with high-density manifolds in $\mathcal{D}_{\text{pre}}$, the default attention distribution satisfies:

$$\mathcal{A}_{\text{param}} \gg \mathcal{A}_{\text{context}}$$

Without active gating, the model's factual memory will dominate the output sequence, ignoring contradictory local prompt rules.

#### Axiom II: Attention Partitioning
To successfully execute Context Learning (CL), there must exist an active attention-decoupling threshold $\kappa > 0$ such that the ratio of local contextual attention to parametric memory attention is strictly bounded from below:

$$\frac{\mathcal{A}_{\text{context}}}{\mathcal{A}_{\text{param}}} \ge \kappa$$

### 2. Theorems of Learnability Boundaries

#### Theorem I: Manifold Incompatibility
Let $\mathcal{M}_{\mathcal{T}}$ be the Riemannian manifold of the teacher model's latent representation space, and let $\mathcal{M}_{\mathcal{S}}$ be the manifold of the student model's representation space. Let $d(\mathcal{M})$ represent the geometric dimension of the manifold.

If the teacher generates a Context-CoT reasoning trajectory $\tau_{\mathcal{T}}$ with topological complexity $K(\tau_{\mathcal{T}})$ that exceeds the dimensional capacity of the student:

$$K(\tau_{\mathcal{T}}) > d(\mathcal{M}_{\mathcal{S}})$$

Then there exists no projection mapping $\phi: \mathcal{M}_{\mathcal{T}} \to \mathcal{M}_{\mathcal{S}}$ that preserves the semantic logical invariants of the reasoning steps. 

*Proof Sketch:* Attempting to force $\mathcal{M}_{\mathcal{S}}$ to fit the high-curvature trajectory $\tau_{\mathcal{T}}$ results in a non-convergent KL divergence:

$$\lim_{t \to \infty} D_{\text{KL}}\left(P_{\mathcal{T}}(y \mid x, \tau_{\mathcal{T}}) \;\parallel\; P_{\mathcal{S}}(y \mid x, \phi(\tau_{\mathcal{T}}))\right) = \infty$$

This manifesting as structural syntax collapse during fine-tuning (e.g., repeating punctuation, failing formatting tags, or random formatting execution failures).

#### Theorem II: The Correctness-Learnability Disjunction
Let $\tau$ be a reasoning trajectory generated by a frontier teacher. Let $\text{Correct}(\tau) \in \{0, 1\}$ be the objective correctness of the trajectory, and let $\text{Learnable}(\tau \mid \mathcal{S}) \in [0, 1]$ be the student's capacity to absorb the sequence.

$$\exists \tau \quad \text{such that} \quad \text{Correct}(\tau) = 1 \quad \text{and} \quad \text{Learnable}(\tau \mid \mathcal{S}) \approx 0$$

*Justification:* Peking/Tsinghua empirical evaluations show that 4,000 *learnable* (filtered, geometrically simplified) trajectories yielded a $4\%$ absolute performance gain on CL-Bench, whereas 4,000 *mathematically complex* (unfiltered, raw frontier) trajectories caused a performance degradation due to distillation misalignment.
